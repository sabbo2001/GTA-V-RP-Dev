Auto Ecole


Bienvenue a L'auto Ecole


Nous exigeons que tous nos citoyens passent par un test théorique avant d'obtenir un permis de conduire valide.
Prenez votre temps et répondez avec le bon sens.

Code de la route:
- Le test écrit coûte 200 €, il n'y a pas de remboursement si vous échouez.
- Ne vous inquiétez pas, le Centre D'auto Ecole permettra des prêts, veillez à ne pas tomber dans une grande dette
- Si vous ne réussissez pas le test une fois, vous ne pouvez pas le reprendre tout de suite, revenir un autre jour et réessayer.

Test de Conduite
- Le test de conduite coûte 500 €, comme auparavant, ce paiement n'est pas remboursé si vous échouez le test.
- Le but principal du test de conduite est l'attention portée aux détails, faites attention à votre environnement!
